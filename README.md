# Agytorna
Többnyelvű kvízjáték amelyben orosz, német, angol és magyar nyelven kaphatsz szavakat vagy mondatokat. Válaszd ki a jó megoldást az előre megadottak közül!

<img width="515" height="203" alt="image" src="https://github.com/user-attachments/assets/34130b95-1406-4578-badc-8278772d27f5" />
<p></p>
Az alkalmazás fejlesztése C++ nyelven történik, amely biztosítja a gyors és hatékony működést.
A felhasználói felület alkalmazásként valósul meg, C++ grafikus keretrendszer Qt keretrendszer segítségével.
A program a SQlite adatbázissal fog működni amely tartalmazza a felhasználókat valamint az ők statisztikájukat.
